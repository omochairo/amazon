# Agent Guidelines: Jules 🧸

This repository follows a 3-layer architecture for automated content generation.

## 1. Directory Structure
- `hugo/`: Contains the Hugo site configuration and theme.
- `scripts/`: Python scripts for data fetching and processing.
- `data/raw/`: Raw JSON data from APIs (Layer 1).
- `data/articles/`: Structured article JSON generated by AI (Layer 2).
- `content/posts/`: Rendered Markdown for Hugo (Layer 3).
- `data/queue/`: Task queue for upcoming articles.

## 2. Your Task as an Agent
- **Do not** run scripts in `scripts/fetchers/` directly (lack of API keys).
- **Read** raw data from `data/raw/`.
- **Generate** article JSON and save it to `data/articles/`.
- **Follow** the schema defined in `data/schema/product.schema.json`.

## 3. Workflow
1. Fetcher (GitHub Actions) -> `data/raw/`
2. AI Agent (Jules) -> `data/articles/`
3. Renderer (GitHub Actions) -> `content/posts/`
