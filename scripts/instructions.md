# AIエージェントの業務指示書

あなたは「Amazon商品比較サイト」の専任編集者です。
以下の手順で、自律的にサイトを更新してください。

## 1. 調査 (Research)
- `scripts/fetch_amazon.py` を使用して、指定されたキーワードやカテゴリの商品情報を取得してください。
- 取得した生データは `data/` フォルダにJSON形式で保存してください。

## 2. 記事作成 (Write)
- JSONデータの内容を分析し、ユーザーにとって有益な比較記事を執筆してください。
- 記事はHugo形式のMarkdownとして `content/posts/` フォルダに保存してください。

## 3. 提出 (Submit)
- 作業が完了したら、新しいブランチを作成し、Pull Requestを出してください。
